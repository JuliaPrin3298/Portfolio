### Portfólio

Portfólio com projetos em Desenvolvimento de Software, Ciência de Dados e Design de Jogos.  
O desenvolvimento deste portfólio web serve tanto para apresentar minhas habilidades quanto para o meu próprio aprendizado, permitindo que interessados em meus serviços avaliem os projetos que já desenvolvi.

---

## Objetivos

Com este portfólio, busco alcançar minha primeira oportunidade de trabalho na área da tecnologia, além de demonstrar minha evolução técnica ao longo do tempo.  
Ao navegar pelo site, será possível entender meu estilo de trabalho, minhas áreas de interesse e o que gosto de desenvolver no meu tempo livre.

---

## Funcionalidades e Tecnologias

O portfólio contém as seguintes abas: **Home**, **About Me**, **Portfolio**, **Skills**, **Certificates** e **Contact**.  
A escolha de desenvolver o portfólio fora da minha língua nativa (PT-BR) deve-se ao objetivo de demonstrar minha fluência na língua inglesa.

Tecnologias utilizadas no projeto:

| Tecnologia      | Finalidade                         |
|-----------------|------------------------------------|
| Figma           | Prototipação e design              |
| HTML            | Estruturação do site               |
| CSS             | Estilização do portfólio           |
| Bootstrap       | Responsividade                     |
| Google Fonts    | Tipografia utilizada no projeto    |
| JavaScript      | Animações de interface, tooltips dinâmicos e lógica de envio de emails no formulário de contato     |

O projeto conta com um conjunto de pastas destinadas à organização, incluindo o diretório de imagens (`images`),uma pasta de apêndices(`appendix`) que contém os arquivos `.pdf` e `.rar` e outra voltada para os arquivos de estilo(`css`), garantindo melhor organização e manutenção do código.
